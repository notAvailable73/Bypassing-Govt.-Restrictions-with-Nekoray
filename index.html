Step 1:

Download NekoRay from GitHub or GoogleDrive or Telegram

Step 2:

Extract the folder and run nekoray.exe as administrator.

Step 3:

Choose any of the two options from here

Step 4:

After the app opens Click on Preferences -> Groups

Step 5:

Click on New Group and then follow the instructions below or do as shown in the image

  • Name: Give the group a name.
  • Change the Type from Basic -> Subscription
  • Paste this link in the URL option: https://raw.githubusercontent.com/clientprovpn/v2/main/Daily

Note: You can find other proxies or subscription links from these repos - (I haven't tested them)

  • V2ray-Configs
  • ClientProVPN

Step 6:

Click on Update Subscription and wait till it shows a number beside the Subscription (marked as 2 in the image)

  • If it shows a number then close the window.
  • If it says Requesting subscription Name error: Operation canceled then try connecting a VPN first and then update the list.
  • If connecting VPN doesn't help use this link:
    • https://raw.githubusercontent.com/barry-far/V2ray-Configs/main/Splitted-By-Protocol/trojan.txt
    • Or use any other links from this GitHub Repo

Step 7:

Select the Group you have just created from the main window.

Step 8:

Click on Server - Current Group - TCP Ping

Step 9:

The Final Step

  • Check the Tun Box (Important!)
  • Right Click on a Server with good ping and click on Start
  • If it shows upload and download speeds as shown in step 3 you're good to go.

Notes:

  • Not all servers will work for all ISPs
  • You will have to find the server that works for you through trial and error
  • From my experience the Trojan type servers are working for most BD ISPs and the servers with Freenet in the name aren't working
